LightSwitch

Lightweight keyboard layout switcher for Windows

Switch keyboard layouts with a single key tap (e.g. tap LCtrl for English, RCtrl for Ukrainian), and convert already-typed text from one layout to another with a hotkey — without losing the original clipboard.

Features

  • Single-key switching. Bind layouts to standalone modifiers (LCtrl, RCtrl, LAlt) or combinations.
  • Text conversion hotkey. Convert selected text — or just the last word — between layouts.
  • Cyclic conversion across all installed layouts, with automatic source-layout detection.
  • Lean. Single ~2 MB executable, no installer, no dependencies.
  • Privacy-respecting. No telemetry, no network.

Settings

System requirements

Windows 10 or later (x86-64).

Per-user autostart, no admin rights required.

Configuration is stored at %APPDATA%\LightSwitch\config.toml.
